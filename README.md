# Urban Mobility & Economic Productivity Analysis (LATAM)

## Project Description
This project analyzes the relationship between urban mobility and economic productivity across cities in Latin America.

## Objective
To explore how mobility patterns may influence economic activity and identify trends that can support data-driven decision-making.

## Dataset
The dataset includes information related to:
- Urban mobility indicators
- Economic activity metrics
- City-level data across LATAM

## Tools Used
- Python
- Pandas
- numpy
- seaborn
- matplotlib
- Jupyter Notebook

## Analysis Process
- Data cleaning and preprocessing
- Handling missing values and duplicates
- Data structuring and integration
- Exploratory Data Analysis (EDA)

## Key Insights
- Mobility patterns show potential relationships with economic activity in major cities
- Variations across cities suggest differences in infrastructure and economic dynamics
- Clean and structured datasets enable deeper analytical exploration

## Conclusion
The analysis highlights the importance of mobility data in understanding economic productivity and supports further exploration for business and urban planning decisions.

## How to Run
1. Download the dataset
2. Open the notebook in Jupyter Notebook
3. Run all cells
